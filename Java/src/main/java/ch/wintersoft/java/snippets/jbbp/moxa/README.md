# Moxa IP Address Report

This package contains a custom `JBBPCustomFieldTypeProcessor` to retrieve the IP Address report of a Moxa embedded
serial to Ethernet module.

For additional information about Moxa modules check out the modules and corresponding documentation
<https://www.moxa.com/en/products/industrial-edge-connectivity/serial-device-servers/serial-embedded-modules>

Find the `JBBP (Java Binary Block Parser)` library on [GitHub](https://github.com/raydac/java-binary-block-parser)
